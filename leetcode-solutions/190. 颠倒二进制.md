## 190. 颠倒二进制

### 思路

和颠倒十进制有点类似

| 操作             | 十进制  | 二进制 |
| ---------------- | ------- | ------ |
| 取个位           | `n%10`  | `n&1`  |
| 整体向高位挪一位 | `n*10`  | `n<<1` |
| 整体向低位挪一位 | `n//10` | `n>>1` |

```cpp
class Solution {
public:
    uint32_t reverseBits(uint32_t n) {
        int i=32;
        uint32_t res=0;
        while(i--){
            res<<=1;//类似于十进制的res*10+n%10
            res+=n&1;
            n>>=1;//类似于十进制的n//=10          
        }
        return res;
    }
```
